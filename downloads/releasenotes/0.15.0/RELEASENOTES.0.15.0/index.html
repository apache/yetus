Apache Yetus 0.15.0 Release Notes

These release notes cover new developer and user-facing incompatibilities, important issues, features, and major improvements.

  • YETUS-1193 Major Update to Jammy Jellyfish Ubuntu LTS

Beyond the operating system, the following components have been updated:

  • hadolint: 2.10.0 -> 2.12.0
  • buf: 1.8.0 -> 1.9.0
  • bats: 1.8.0 -> 1.8.2
  • golangci-lint: 1.50.0 -> 1.50.1
  • maven: 3.8.5 -> 3.8.6
  • ansible: 5.5.0 -> 6.6.0
  • ansible-lint: 6.0.2 -> 6.8.7
  • codespell: 2.2.1 -> 2.2.2
  • detect-secrets: 1.2.0 -> 1.4.0
  • pylint: 2.13.4 -> 2.15.6
  • yamllint: 1.26.3 -> 1.28.0
  • bundler: 2.3.22 -> 2.3.26
  • rubocop: 1.26.1 -> 1.39
  • jshint: 2.13.4 -> 2.13.6
  • markdowncli-lint: 0.31.1 -> 0.32.2

Additionally:

  • Python is now 3.10-based
  • docker-compose python version has been replaced with the go code plug-in
  • YETUS-1222 Major smart-apply-patch EXTRA_ARGS should be an array

The undocumented EXTRA_ARGS environment variable is now expected to be an array.
